Welcome to the Ada Theme Demonstration

This is a static HTML-File which illustrates all Modules, which can be utilized by Authors within this theme. There are a few things you should know about Ada.

  1. This Theme 100%e2r which means, it focuses above all on readability.
  2. This Theme is inspired by Containerist which means, it abandons all Concepts of Sidebars in favour of a stack of Contianers.
  3. This Theme is guided by the SMACSS principles, which means, it tries to make everything as reusable as possible.
  4. This Theme is constructed with an 18 Column Grid of 1118 Pixel width, which is abandonded in favour of a fluid Grid on displays smaller than 1118 Pixel.

module.text

The Text Module is mostly used for the main content of Contents, like Posts and Pages. Although it is 9 colums wide, which is half of the page and covers the columns 4 - 12, which leave 3 columns to the left and 6 columns to the right. Within the Text module all reasonalbe Content-Element of HTML could be used like …

  • Ordered and unordered Lists
  • Definition Lists
  • Blockquotes
  • Images

Marginalia and Apostille

Also there are two specialized Modules, that only exist an work within the text module:

.marginalia
This module moves a given content to the left 3 columns of the text. Images within the marginalia are shrinked to a maximal width of 163 pixels.
.apostille
This module moves given content to the right 6 columns of the text. Images within the apostille are shrinked to a maximal width of 346 pixels.
